# English Conditional Sentences – Interactive Learning Website

An interactive website to learn and practice **English Conditional Sentences** through theory explanations and a timed quiz.

This project helps learners understand the **Zero, First, Second, and Third Conditional** using clear explanations, examples, and a quiz with instant feedback.

---

## 🌐 Live Website

👉 https://aniketfagare11-lang.github.io/EnglishgDemo1-Website/

---

## 📚 Features

* Clean and modern UI
* Explanation of **4 Conditional Types**
* Interactive **20-question quiz**
* **30-second timer** per question
* Instant feedback and explanations
* Score tracking
* Animated results dashboard
* Dark mode toggle
* Responsive design (mobile friendly)

---

## 🧠 Conditional Types Covered

1. **Zero Conditional** – General truths and scientific facts
2. **First Conditional** – Real and possible future situations
3. **Second Conditional** – Hypothetical or imaginary situations
4. **Third Conditional** – Imagined past situations and regrets
